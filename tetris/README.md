# TETRIS

The original tetris following the tetris guide to bulid an official tetris

with [SRS](https://tetris.fandom.com/wiki/SRS)


## Controlls
- UP -> Rotate CW
- DOWN -> soft Drop
- LEFT -> left
- RIGHT -> right
- X -> rotate CW
- Z -> rotate CCW
- H/C -> hold piece
- G -> enable/disable GHOST
- L -> enable/disable GRID
- P -> pause
## CONTROLLS TESTING
- T -> test SRS
- W -> move up
